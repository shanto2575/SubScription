# DigiTools
A modern web application for managing and purchasing digital tools and products with a beautiful UI and interactive features.
---------------------------------------------
##  Technologies Used

❄️React
❄️JavaScript (ES6+)
❄️TailwindCSS
❄️Lucide-React (Icons)
❄️JSON (Product & Cart Data)

---------------------------------------------

##Key Features

1. **Responsive Navbar & Cart:** Fixed navbar with cart icon showing total items.  
2. **Product Cards & Toggle:** Display products in a 3-column layout; toggle between Products and Cart view.  
3. **Cart Functionality:** Add/remove products, view total, and “Proceed to Checkout” to clear cart.
